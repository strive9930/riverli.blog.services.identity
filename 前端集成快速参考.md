## 🔍 关键代码片段

### 登录响应数据结构

```typescript
interface LoginResponseData {
  token: string;              // JWT Token
  expiration: string;         // 过期时间
  userId: string;            // 用户ID
  username: string;          // 用户名
  nickname?: string;         // 昵称
  email?: string;            // 邮箱
  avatar?: string;           // 头像
  roles: RoleInfo[];         // 角色列表 ⭐
  permissions: string[];     // 权限列表 ⭐
  menus: MenuTreeItem[];     // 菜单树 ⭐
  isAdmin: boolean;          // 是否管理员 ⭐
  createdAt?: string;        // 创建时间
}
```

### User Store 关键变更

```typescript
// 新增的计算属性
const menus = computed(() => userInfo.value?.menus || [])
const isAdmin = computed(() => userInfo.value?.isAdmin || false)

// 简化的登录方法
const login = async (credentials) => {
  const response = await service.post('/auth/login', credentials);
  
  if (response.success && response.data) {
    // 一次性保存所有数据
    accessToken.value = response.data.token;
    userInfo.value = {
      id: response.data.userId,
      username: response.data.username,
      roles: response.data.roles,
      permissions: response.data.permissions,
      menus: response.data.menus,
      isAdmin: response.data.isAdmin,
      // ...其他字段
    };
    
    localStorage.setItem('access_token', accessToken.value);
    localStorage.setItem('user_info', JSON.stringify(userInfo.value));
    
    return { success: true };
  }
}
```

## 🐛 故障排查

### 问题1: 登录后菜单不显示

**检查点:**
```typescript
// 1. 确认登录响应包含 menus
console.log('Menus:', response.data.menus)

// 2. 确认 userInfo 正确保存
console.log('UserInfo:', userInfo.value)

// 3. 确认路由生成被调用
console.log('Routes loaded:', permissionStore.isRoutesLoaded)
```

### 问题2: 权限判断失败

**检查点:**
```typescript
// 1. 确认 permissions 数组有值
console.log('Permissions:', userInfo.value?.permissions)

// 2. 确认权限码格式正确
console.log('Permission sample:', userInfo.value?.permissions[0])

// 3. 检查权限指令使用
<div v-permission="'users.view'">...</div>
```

### 问题3: 刷新后丢失状态

**检查点:**
```typescript
// 1. 确认 localStorage 中有数据
console.log('Stored token:', localStorage.getItem('access_token'))
console.log('Stored user:', localStorage.getItem('user_info'))

// 2. 确认 initUserState 被调用
// 在 main.ts 或 App.vue 的 onMounted 中
await userStore.initUserState()
```
